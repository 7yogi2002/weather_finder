# 🌦️ Weather Finder Application

## 📌 Project Description
This is a simple Weather Finder web application that allows users to search any city and get real-time weather details like temperature, humidity, wind speed, and weather conditions using the OpenWeatherMap API.

The project is fully responsive and built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- 🌍 Search weather by city name  
- 🌡️ Show temperature in °C  
- 💧 Show humidity percentage  
- 🌬️ Show wind speed  
- ☁️ Weather condition icons (Clear, Clouds, Rain, Drizzle, Mist)  
- ❌ Error handling for invalid city  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API  

---

## 🔑 API Used
OpenWeatherMap API – used to fetch real-time weather data based on city name.

---

## 📸 How It Works
1. User enters a city name in the search box  
2. Clicks the search button  
3. App fetches data from OpenWeatherMap API  
4. Displays:
   - City name  
   - Temperature  
   - Humidity  
   - Wind speed  
   - Weather icon based on condition
